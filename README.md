KaffeShop

Ett enkelt fullstack-projekt för en kaffeshop byggt med MySQL, Node.js och en frontend-klient.

Om projektet
• Bläddra bland produkter
• Lägga beställningar
• Hantera användare och profiler
• Administrera kategorier och lager


Databasen innehåller
• Primär- och främmande nycklar
• 1–1, 1–M och M–M relationer
• Constraints och index


Teknik
• MySQL
• Node.js
• Express
• JavaScript


Installation
• Skapa databasen: CREATE DATABASE kaffeshop;
• Kör schema- och seed-filerna.
• Installera beroenden: npm install
• Starta servern: node index.js
